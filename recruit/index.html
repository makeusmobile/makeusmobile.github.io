2018년 상반기산업기능요원 (병역특례 보충역) 채용채용기간
2018년 2월 ~ 2018년 4월

Asia No.1 콘텐츠 & 미디어 모바일 방송국
메이크어스는 모바일, SNS, 그리고 개인화 혹은 또 다른 무엇으로 변화하는 미디어 시대에 가장 최적화된 콘텐츠를 만드는 크리에이티브 그룹입니다. 
우리는 엔터테인먼트, 음식, 여행, 뷰티, 스타일, 음악, 예능 등 다양한 콘텐츠를  우리만의 방법으로 만들고, 그 공감과 정보를 즐거움이란 언어로 나누고 있습니다.
메이크어스는 모바일 세대의 라이프 스타일을 반영한 미디어와 커머스, 그리고 엔터테인먼트 분야의 글로벌 비즈니스를 지향합니다.

공통사항 고용형태
공통 지원자격

* 4급 보충요건 충족에 문제가 없는 분 (당해 년도 대상, 아래 조건들 중 하나 이상 충족)

공통 우대사항
  • 1. GitHub 등에 개인 프로젝트 코드를 가지고 있는 분 우대
  • 2. GitHub 에 공개된 라이브러리 코드를 읽고 이해하여 풀 리퀘스트를 날려보신 분 우대
  • 3. 영문 문서 (Stackoverflow, MDN)를 읽는 데에 거부감이 없으며 가짜 정보를 걸러낼 수 있는 분 우대
  • 4. 소셜 미디어 (Facebook, Instagram, YouTube, Twitter 등) 사용 경험이 있으며 Page 등 자체 브랜드 운영 경험 우대 
  • 5. Git을 써봤고 Git Flow를 설명 할 수 있는 분 우대 (Pull Request 개수 당 가산점 부여)
  • 6. Jira를 써봤고 Scrum 보드와 Kanban 보드의 차이를 설명 할 수 있는 분 우대
  • 7. Confluence를 써봤고 Markdown 문법에 능숙하신 분 우대 
  • 8. AWS를 활용한 서비스를 개발해 봤고 EC2 등 AWS 서비스 설정이 가능하신 분 우대
  • 9. 빌드, 테스트, 배포 자동화 경험이 있으며 Jenkins / Nexus 등 설정 가능 하신 분 우대
  • 10. Facebook graph API, Facebook Marketing API 사용 경험 우대
  • 11. YouTube data API v3, YouTube analytics API 사용 경험 우대
  • 12. YouTube Certification 보유 (종류 무관) 시 우대 

4. 분야별 지원 자격 및 우대사항

모집부문 모집내용
프론트 엔드 지원자격
  • - HTML, CSS, Javascript 기본 지식을 보유하신 분
  • - ES2015 이후의 문법이 어색하지 않고, Promise는 익숙하신 분
  • - HTLM5 태그를 기술 할 수 있고, CSS 만을 이용해 레이아웃을 잡는 것이 가능하신 분
  • - JQuery 없이도 자유로운 화면 개발이 가능 하신 분
  • - Browser Developer Tool을 이용해 Debuging이 가능 하신 분
  • - Web API (aka.RESTful API) 클라이언트 개발 경험이 있으신 분
우대사항
  • - JavaScript Framework(Vue, Angular, 혹은 Ember 등)을 이용해 SPA 앱을 만들어 보신 분 우대
  • - ReactJS를 써봤고 Redux를 연동할 수 있는 분 우대
  • - React environment (Redux, Webpack, Babel 등)에 능숙하신 분 우대
백엔드 지원자격
  • - Java 8, Spring 4.x 이상 기반 개발이 가능하신 분
  • - MVC 패턴을 이해하고 Spring MVC 사용에 무리가 없으신 분
  • - MySQL 등 RDBMS 사용 경험이 있고 기본적인 SQL 작성이 가능하신 분
  • - 리눅스 기본 명령어를 5가지 이상 말할 수 있고 터미널 기반 작업이 가능하신 분
  • - Http Method별 차이를 이해하고 RESTful API 기반 서비스 개발 경험이 있으신 분
우대사항
  • - Spring의 Proxy 패턴을 이해하고 Spring Bean에 대해 설명할 수 있는 분 우대
  • - @Transactional 어노테이션을 이해하고 Propagation에 따른 차이점을 설명 할 수 있는 분 우대 
  • - JPA 또는  MyBatis 등 ORM 사용 경험과 도메인 모델링 경험 있으신 분 우대
  • - RDBMS의 Isolation Level에 따른 차이를 설명할 수 있는 분 우대 
  • - Cassandra, HBase, CouchBase, MongoDB 등 NoSQL 경험 우대
  • - 리눅스 기반 nginx 또는 Tomcat 을 설치 할 수 있고 설정 및 튜닝 가능하신 분 우대
  • - SELinux를 이해하고 관리 할 수 있는 분 우대 
  • - Go에 관심이 있고 Go를 이용한 개발 경험이 있는 분 우대
  • - Elixir에 관심이 있고 Elixir를 이용한 개발 경험이 있는 분 우대
